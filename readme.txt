Porjet tutoré réalisé en deuxième année de licence inofrmatique, nous étions 4 étudiants et avions 12 semaines.

Pour lancer le projet, se rendre dans le dossier correspondant à votre système d'exploitation.
Puis double cliquez sur l'executable.
Si l'execution est impossible, se rendre dans le dossier précédent, ouvrir un terminal et tapez python (ou python3) L2J2_RP.py
Le programme s'exécuterait alors.
Si, lors de l'execution du programme en utilisant python, notamment python 2.X, vous obtenez une erreur comme celle-ci : ImportError: No module named concurrent.futures.process
, veuillez ouvrir un terminal et taper :
pip install futures
Relancez ensuite le programme python.
